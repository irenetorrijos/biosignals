Band Pass Filter Design¶

  • Design a band pass filter. Use Butterworth filter of order 4 with cutoff frequencies equal to 0.5Hz and 40Hz.
  • Design another butterworth band pass filter but this time use an order of 2 with cutoff frequencies equal to 0.5Hz and 40Hz
  • Plot the frequency response of both filters and explain your observations
* Hint use the following three functions scipy.signal.butter() and scipy.signal.freqz()

  • Important to use the scipy.signal.freqz() function and NOT the scipy.signal.freqs() function.

      scipy.signal.freqz() is for digital filters which is what we want.

https://docs.scipy.org/doc/scipy/reference/signal.html#filtering link to info about scipy.signal functions

In [ ]:
# Filter order 4
b4, a4 = sg.butter(4, Wn = (0.5, 40), btype='bandpass', fs = fs)

# Filter order 2
b2, a2 = sg.butter(2, Wn = (0.5, 40), btype='bandpass', fs = fs)


#Frequency response of the filters
w4, h4 = sg.freqz(b4, a4, worN=8000)
w2, h2 = sg.freqz(b2, a2, worN=8000)
In [ ]:
# Plot the frequency response
plt.figure(figsize=(12, 7))
plt.plot(0.5 * fs * w4 / np.pi, np.abs(h4), label=f'Order {4}')
plt.plot(0.5 * fs * w2 / np.pi, np.abs(h2), label=f'Order {2}')
plt.title("Frequency Response of Butterworth Bandpass Filters")
plt.xlabel("Frequency (Hz)", fontsize=16)
plt.ylabel("Magnitude", fontsize=16)
plt.grid(True)
plt.legend()
plt.show()

Observations:¶

Both filters start at almos 0 (0.5 Hz) and start droping around 40 Hz, the cutoff frequencies that were given. However, and according to theory, it can be seen that the higher order filter, has a steeper slope, which makes it a steeper filter, making it a more accurate bandpass filter.

Observations:¶

There are 2 main changes in the filtered signal compared with the raw signal:

  • Less noise, the signal is clearer with less peaks that come from noise and artifacts. The QRS complex, P and T waves are preserved.
  • Centered at 0 Hz, one response for the bandpass filter. This means that the frequencies capable of passing this filter are then centered at 0 Hz, to simplify the interpretation. The now 0 was the baseline amplitude.
